# 🕒 Python Mini Project - Digital Clock

A simple yet elegant **Digital Clock application** built using Python and Tkinter. This project displays the current system time in real-time with a clean graphical interface.

---

##  Features

* Real-time clock display (updates every second)
* Supports both 24-hour and 12-hour (AM/PM) formats
* Simple and attractive GUI using Tkinter
* Lightweight and easy to run

---

##  Technologies Used

* Python
* Tkinter (GUI library)
* Time module

---

##  Tools Used

* PyCharm IDE (for writing and running the code)
* Python Interpreter

---

##  How to Run

1. Install Python
2. Open the project in PyCharm
3. Run the file:

python digital_clock.py
```

---

##  Output

Displays a live digital clock with a black background and cyan text.
